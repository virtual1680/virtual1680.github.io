介绍 #

qv-vue 是一个基于 Element Plus 框架 低代码前端框架，它使用 JSON 配置来生成页面，可以减少页面开发工作量，极大提升效

  • 💋 支持按需加载 / 全量加载
  • 💪 完善的组件提示功能 三斜线方式

TIP

你可以跳过本章节，直接进入下一章节，了解开发 UI 组件库的核心步骤

技术依赖 #

  • vue3
  • vite
  • typeScript
  • element-plus
